## 使用内置函数

```go title="Go"
var integer16 int16 = 127
var integer32 int32 = 32767
fmt.Println(int32(integer16) + integer32)
```

## 使用 strconv 包

### Atoi

`Atoi()` 将 **string** 转为 **int**。

### Itoa

`Itoa()` 将 **int** 转为 **string**。
