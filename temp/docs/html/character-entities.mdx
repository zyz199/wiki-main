## 空格

浏览器会将添加的空格**压缩为一个**。文本前后的空格**会被忽略**，内部的连续空格**只会算一个**。

### 插入连续空格

- `&nbsp;` - 一个字符的半角的**不断行**的空格
- `&ensp;` - 一个字符的半角的空格
- `&emsp;` - 两个字符的全角的空格
- `&thinsp;` - 小于一个字符的空格

## 常用字符实体

- &lt; 小于号 - `&lt;` (less than)
- &gt; 大于号 - `&gt;` (greater than)
- &amp; And - `&amp;` (ampersand)
- &quot; 双引号 - `&quot;` (quotation)
- &apos; 单引号 - `&apos;` (apostrophe)
- \_ 下划线 - `&lowbar;`
