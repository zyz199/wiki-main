## DRY

> **Don't Repeat Yourself.**

## KISS

> **Keep It Simple, Stupid.**

## 命名规范

### 变量名

不可以用**<TH>空格</TH>**、**<TH>连词符</TH>**(-)、**<TH>下划线</TH>**(\_) 等将变量名隔开。

### Camel Notation (Camel Case)

**第一个单词的首字母小写**，后面的其他单词首字母全部大写。

例：firstName、lastName

### Pascal Notation (Pascal Case)

**每一个单词的首字母都采用大写**。

例：FirstName、LastName
