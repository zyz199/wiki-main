## 块级元素

HTML 中的块级元素如 `<div>`，`<table>`，`<pre>` 和 `<p>`，请在它们前后插入<TH>空白行</TH>来和其他内容分隔开。请不要使用<TH>制表符</TH>或<TH>空格</TH>对 HTML 元素进行缩进, 否则会影响其格式化.

我们<TH>无法在 HTML 块级元素内部使用 Markdown 语法</TH>。例如 `<p>*italic* _italic_ and **bold**</p>` 将不会工作。
