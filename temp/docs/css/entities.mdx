<p>
  <TH>HTML 字符实体不可以直接在 CSS 中使用</TH>。为了在 CSS 中显示
  HTML字符实体，我们需要使用 CSS 字符实体。
</p>

如下代码不工作，因为在 CSS 中使用了 HTML 字符实体:

```css {2} title="CSS"
h2:after {
  content: "&#36;";
}
```

改用 **CSS 字符实体**，即可正确显示：

```css {2} title="CSS"
h2:after {
  content: "\0024"; /* \0024 is $ */
}
```
