## 安装 Go package

从 <Link to="https://go.dev/dl/">Go 官网</Link> 下载 Go package 安装包。

### 验证版本

```bash
brucesong@localhost ~ % go version
go version go1.18.3 darwin/arm64
```

## Go 第三方包代理设置

### 使用 VPN 代理

如果有可以访问国外站点权限的 VPN 代理，设置通过该 VPN 代理访问 **google.golang.org**。

### 使用镜像地址

<p style={{ marginBottom: 4 }}>macOS 下在终端执行如下命令：</p>

```bash
export GOPROXY=https://goproxy.cn
```
