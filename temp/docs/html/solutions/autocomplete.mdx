> 当我们自己实现自动补全 UI 时，往往浏览器默认的自动补全会对我们的样式造成影响，此时可以通过 **`autocomplete`** 属性**禁用浏览器的自动补全**。

## &lt;input&gt; 禁用自动补全

```html title="HTML"
<input type="text" autocomplete="off" />
```

## &lt;form&gt; 禁用自动补全

```html title="HTML"
<form autocomplete="off"></form>
```
