## 变量声明

```go title="Go"
a := "Unused Variable"
```

```bash title="错误信息"
# command-line-arguments
./hello.go:7:2: a declared but not used
```

:::caution 注意
在 Go 中，当你声明一个变量但不使用它时，Go 会抛出错误，而不是像某些其他编程语言一样抛出警告。
:::

`_` 也可以作为一个变量，通常用于<TH>忽略函数的返回值</TH>。
