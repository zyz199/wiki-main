## Stroke

通过 `stroke` 属性**填充边框颜色**。

有如下示例 svg 元素绘制出三条由细到粗的等长黑色线条：

```html title="HTML"
<svg xmlns="http://www.w3.org/2000/svg">
  <g fill="none" stroke="#000">
    <path stroke-width="2" d="M5 20 l215 0" />
    <path stroke-width="4" d="M5 40 l215 0" />
    <path stroke-width="6" d="M5 60 l215 0" />
  </g>
</svg>
```

<svg
  style={{ height: "80px" }}
  xmlns="http://www.w3.org/2000/svg"
  version="1.1"
>
  <g fill="none" stroke="#000">
    <path strokeWidth="2" d="M5 20 l215 0" />
    <path strokeWidth="4" d="M5 40 l215 0" />
    <path strokeWidth="6" d="M5 60 l215 0" />
  </g>
</svg>

<p>
  通过属性选择器选择 <code>stroke</code> 属性，为其定指定 CSS 属性{" "}
  <code>stroke: #ffff00</code> ：
</p>

```css
svg [stroke="#000"] {
  stroke: #ffff00;
}
```

<p style={{ margin: "0" }}>如下图所示，黑色线条变成了黄色线条：</p>
