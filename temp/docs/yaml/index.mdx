> **YAML** (读音 /'jæməl/ ) 是一种**可读性高**的**数据序列化语言**，通常用于编写配置文件，其文件扩展名为 `.yaml` 。

## 基本特性

- 大小写**敏感**
- 使用**缩进**示层级关系
- 缩进不允许使用 `tab`，只允许空格
- 缩进的空格数不重要，只要相同层级的元素**左对齐**即可
- `#` 表示注释
