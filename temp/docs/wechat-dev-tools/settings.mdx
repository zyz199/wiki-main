## 文件保存后自动格式化

在 `settings.json` 中配置：

```json title="JSON"
{
  "editor.formatOnSave": true
}
```

## 插件

### Prettier

配置格式化工具 **Prettier**，在 `settings.json` 中配置：

```json title="JSON"
{
  "[javascript]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode"
  },
  "[json]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode"
  }
}
```
