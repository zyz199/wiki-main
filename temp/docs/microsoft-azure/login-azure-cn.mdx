## 使用不同的 IDE 登录中国区 Azure

<Tabs groupId="IDE">
  <TabItem value="Visual Studio Code">
    在 <strong>设置 - 扩展 - Azure Configuration - Azure Cloud</strong> 选择&nbsp;
     <TH>AzureChina</TH>。<br />
    接着登录 Azure 会出现中国区 Azure 的 URL，输入设备码会跳转到中国区 Azure
    登录页面。
  </TabItem>
  <TabItem value="Visual Studio">⌛️ 完善中...</TabItem>
  <TabItem value="Rider">
    在 <strong>用户主目录/AzureToolsForIntelliJ</strong> 下创建文件 <strong>AadProvider.json</strong>，如果是 <strong>MacOS</strong> 则文件名为 <strong>AuthMethodDetails.json</strong>，并按如下所示更新其内容：

```json
{
  "EnvironmentName": "CHINA"
}
```

<TH>重新启动</TH> IDE后，便可登录中国区 Azure。

如果要切换回国际版 Azure，注释或删除该文件，然后<TH>重新启动</TH> IDE。

  </TabItem>
</Tabs>
