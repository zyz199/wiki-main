可以在 CSS 中引入**自定义字体**，控制字体显示效果。

## JetBrains Mono 字体

```css title="CSS"
@import url("https://fonts.googleapis.com/css2?family=JetBrains+Mono&display=swap");

code {
  color: var(--ifm-color-primary);
  /* 使用 JetBrains Mono 字体 */
  font-family: "JetBrains Mono", monospace !important;
  padding-left: 5px;
  padding-right: 5px;
}
```
