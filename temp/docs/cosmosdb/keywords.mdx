## DISTINCT

`DISTINCT` 关键字用于去除查询中的重复项。

```sql title="SQL"
SELECT DISTINCT VALUE b.name
FROM Books b
```

返回不包含重复项的数组，若有两本书都叫 "JavaScript Tutorial"，那么只会返回一个。

```json title="JSON"
["SQL Server Tutorial", "JavaScript Tutorial"]
```
