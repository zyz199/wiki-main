## 什么是 GUID

**GUID** 即全球唯一的标识符（globally unique identifier），也称 **UUID**（universal unique identifier） ，是一种 16 字节或 128 位数字。目的是唯一地标识某些东西，其在表、数据库和服务器之间是<TH>全球唯一</TH>的。

## 格式

**xxxxxxxx-xxxx-Mxxx-Nxxx-xxxxxxxxxxxx**

例：69121893-3AFC-4F92-85F3-40BB5E7C7E29

## 在 SQL Server 中创建 GUID

使用 `NEWID()` 函数将 Product 表中 Code 字段更新为随机 GUID：

```sql title="SQL Server"
UPDATE Product SET Code = NEWID()
```

## 在 SQL Server 中声明 GUID 类型变量

使用 `UNIQUEIDENTIFIER` 关键字声明 GUID 变量：

```sql
DECLARE @UNI UNIQUEIDENTIFIER;
SET @UNI = NEWID();
SELECT @UNI;
```
