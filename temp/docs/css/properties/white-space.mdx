## 语法

- `normal` - 连续的空白符会被<TH>合并</TH>，换行符会被当作空白符来处理。
- `nowrap` - 连续的空白符会被<TH>合并</TH>，但文本内的换行无效。
- `pre` - 连续的空白符会被<TH>保留</TH>，在遇到换行符或者 **&lt;br&gt;** 元素时才会换行。
- `pre-wrap` - 连续的空白符会被<TH>保留</TH>，在遇到换行符或者 **&lt;br&gt;** 元素时才会换行。
- `pre-line` - 连续的空白符会被<TH>合并</TH>，在遇到换行符或者 **&lt;br&gt;** 元素时才会换行。
- `break-spaces` - 与 `pre-wrap` 的行为相同。

## 浏览器对空格的默认处理

浏览器会将添加的空格压缩为一个。文本前后的空格会被忽略，内部的连续空格只会算一个。
