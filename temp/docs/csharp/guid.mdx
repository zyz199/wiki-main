## 什么是 GUID

**GUID** 即全球唯一的标识符（globally unique identifier），也称 **UUID**（universal unique identifier） ，是一种 16 字节或 128 位数字。目的是唯一地标识某些东西，其在表、数据库和服务器之间是<TH>全球唯一</TH>的。

## 格式

**xxxxxxxx-xxxx-Mxxx-Nxxx-xxxxxxxxxxxx**

例：69121893-3AFC-4F92-85F3-40BB5E7C7E29

## 在 C# 中创建 GUID

使用 `Guid.NewGuid()` 方法在 C# 中生成一个新的 GUID：

```csharp title="C#"
using System;
namespace GUIDTest {
    class MainClass {
        static void Main(string[] args) {
            Guid guid = System.Guid.NewGuid();
            Console.WriteLine(guid); // expected: 0f8fad5b-d9cb-469f-a165-70867728950e
            Console.WriteLine(Guid.NewGuid()); // expected: 7c9e6679-7425-40de-944b-e07fc1f90ae7
        }
    }
}
```
