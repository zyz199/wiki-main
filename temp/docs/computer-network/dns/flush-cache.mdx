## 在浏览器中清除 DNS 缓存

<Tabs groupId="browser">

## 在 macOS 中清除 DNS 缓存

<p style={{ marginBottom: 4 }}>在终端执行：</p>

```bash
sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder
```
